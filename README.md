# Pharmacy Drug Management System

A Java-based Pharmacy Drug Management System developed using JSP, HTML, CSS, Java, and MySQL to manage medicines, inventory, customer orders, user authentication, and pharmacy operations efficiently.

## Project Description

The Pharmacy Drug Management System is a web-based application developed to simplify and manage pharmacy operations efficiently. This project helps manage medicines, inventory, customer purchases, seller operations, authentication, and order processing. The system provides separate functionalities for customers and sellers/vendors, enabling smooth management of pharmacy-related activities.

The application is developed using Java technologies with JSP for dynamic web pages, HTML and CSS for frontend design, and MySQL for database management.

## Features

* User Registration and Login Authentication
* Seller/Vendor Registration and Login
* Add and Manage Pharmacy Products/Medicines
* Drug Inventory Management and Restocking
* Buy Medicines and Place Orders
* Customer Homepage and Seller Homepage
* Payment Handling and Order Processing
* Database Integration using MySQL
* User-Friendly Web Interface using JSP, HTML, and CSS
